Projets

One Piece Guess

Le projet « Guess the Pirate: One Piece Challenge » est un jeu de devinette qui est basé sur l’univers de « One Piece », une série manga populaire. Dans ce jeu, les joueurs doivent deviner le nom du personnage affiché à l’écran en saisissant leur réponse dans un champ de texte à l’image d’un pendu avec un nombre de chance au nombre de sept. J’ai développé ce projet en utilisant les langages HTML, CSS et JavaScript, démontrant ainsi ma maîtrise de ces technologies front-end et liant ma passion pour le manga avec mon futur métier.

Todolist

Cette application est une interface simple et intuitive pour gérer tes tâches quotidiennes. Construite avec React, elle permet d’ajouter, de supprimer et de filtrer les tâches en un clic. Son design épuré est accompagné d’une sauvegarde automatique des données pour ne jamais perdre le fil de tes activités.

Big Data Music

Le projet Big Data Music illustre mes compétences en développement backend et en gestion de données. Il permet aux utilisateurs de rechercher des chansons et de créer des playlists personnalisées, à la manière de Spotify. Développé avec Python (Flask) pour le backend et une base de données SQL optimisée, il combine performance et simplicité. Ce projet reflète ma capacité à gérer des volumes importants de données tout en offrant une expérience utilisateur fluide.
